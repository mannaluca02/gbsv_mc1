Spielregeln: SET

Ziel des Spiels

Bei SET liegen immer 12 Karten offen auf dem Tisch. Alle Spielenden suchen gleichzeitig nach einem SET, also nach einer passenden Kombination aus 3 Karten.

Wer ein korrektes SET findet, erhält die 3 Karten als Gewinn. Danach wird die Auslage wieder auf 12 Karten ergänzt. Am Ende gewinnt, wer die meisten SETs gesammelt hat.

⸻

Spielmaterial

Das Spiel besteht aus 81 Karten. Jede Karte ist einzigartig und unterscheidet sich durch 4 Eigenschaften:

Eigenschaft	Möglichkeiten
Menge	1, 2 oder 3 Symbole
Farbe	rot, violett oder grün
Form	Raute, Balken oder Welle
Füllung	leer, halb oder voll

⸻

Was ist ein SET?

Ein SET besteht immer aus 3 Karten.

Für jede der 4 Eigenschaften muss gelten:

Die Eigenschaft ist bei allen 3 Karten entweder genau gleich oder völlig verschieden.

Das gilt für:

1. Menge
2. Farbe
3. Form
4. Füllung

Nur wenn alle 4 Fragen mit "Ja" beantwortet werden können, ist es ein SET.

⸻

Die 4 Kontrollfragen

Bei drei Karten fragt man:

1. Menge

Sind die Mengen genau gleich oder völlig verschieden?

Beispiel:

* alle drei Karten haben 1 Symbol → gleich
* eine Karte hat 1 Symbol, eine 2 Symbole, eine 3 Symbole → verschieden
* zwei Karten haben 2 Symbole und eine Karte hat 3 Symbole → kein SET

⸻

2. Farbe

Sind die Farben genau gleich oder völlig verschieden?

Beispiel:

* alle drei Karten sind rot → gleich
* eine Karte ist rot, eine violett, eine grün → verschieden
* zwei Karten sind rot und eine ist grün → kein SET

⸻

3. Form

Sind die Formen genau gleich oder völlig verschieden?

Beispiel:

* alle drei Karten zeigen Rauten → gleich
* eine Karte zeigt eine Raute, eine einen Balken, eine eine Welle → verschieden
* zwei Karten zeigen Wellen und eine Karte zeigt einen Balken → kein SET

⸻

4. Füllung

Sind die Füllungen genau gleich oder völlig verschieden?

Beispiel:

* alle drei Karten sind leer → gleich
* eine Karte ist leer, eine halb gefüllt, eine voll gefüllt → verschieden
* zwei Karten sind leer und eine ist voll → kein SET

⸻

Merksatz

Sind zwei Karten bei einer Eigenschaft gleich und die dritte Karte ist anders, dann ist es kein SET.

Beispiel:

* 2 Karten rot, 1 Karte grün → kein SET
* 2 Karten leer, 1 Karte voll → kein SET
* 2 Karten mit 3 Symbolen, 1 Karte mit 1 Symbol → kein SET

⸻

Spielaufbau

1. Alle 81 Karten werden gut gemischt.
2. Die Karten werden als verdeckter Stapel bereitgelegt.
3. 12 Karten werden offen ausgelegt.
4. Die 12 Karten werden meistens als 3 x 4 Raster angeordnet.

Die drei Karten eines SETs müssen nicht nebeneinanderliegen. Sie können beliebig in der Auslage verteilt sein.

⸻

Spielablauf

Alle Spielenden suchen gleichzeitig nach einem SET.

Wer ein SET gefunden hat:

1. ruft laut und deutlich "SET!"
2. zeigt sofort die 3 Karten
3. die anderen überprüfen, ob es wirklich ein SET ist

⸻

Wenn das SET stimmt

Wenn die 3 Karten ein korrektes SET bilden:

1. Die Person nimmt die 3 Karten aus der Auslage.
2. Sie legt die Karten verdeckt vor sich ab.
3. Die Auslage wird mit 3 neuen Karten vom Stapel wieder auf 12 Karten ergänzt.
4. Das Spiel geht sofort weiter.

Jede gewonnene Karte zählt als Punkt oder man zählt am Ende einfach die gesammelten SETs.

⸻

Wenn das SET nicht stimmt

Wenn die 3 Karten kein SET bilden:

* Das Spiel geht normal weiter.
* Die Person, die falsch gerufen hat, darf so lange nicht mitspielen, bis eine andere Person ein korrektes SET gefunden hat.

⸻

Kein SET in der Auslage?

Es kann vorkommen, dass unter den 12 Karten kein SET vorhanden ist oder dass längere Zeit niemand ein SET findet.

Dann werden 3 weitere Karten vom Stapel dazugelegt.

Die Auslage hat dann kurzfristig 15 Karten.

Nach dem nächsten gefundenen SET wird die Auslage aber nicht wieder auf 15 ergänzt, sondern bleibt wieder bei 12 Karten.

⸻

Spielende

Das Spiel endet, wenn:

1. alle Karten vom Stapel aufgebraucht sind und
2. in der Auslage kein SET mehr vorhanden ist.

Danach zählen alle Spielenden ihre gesammelten Karten oder SETs.

Es gewinnt, wer die meisten Karten beziehungsweise SETs gesammelt hat.

⸻

Varianten

SET'S EASY

Diese Variante eignet sich für Anfängerinnen, Anfänger und jüngere Kinder.

Dabei wird nur mit den 27 Karten einer Farbe gespielt.

Dadurch fällt die Eigenschaft Farbe weg. Das Spiel wird einfacher und übersichtlicher.

Die Auslage besteht dann aus 3 x 3 Karten.

Alle anderen Regeln bleiben gleich.

⸻

SET'S FOREVER

Diese Variante ist ein Turnierspiel ab 3 Spielenden.

Das Spiel geht über so viele Durchgänge, wie Personen mitspielen.

Abwechselnd übernimmt jede Person einmal die Rolle des SET-Masters.

Der SET-Master:

* mischt die Karten
* legt die Karten aus
* ergänzt nach einem SET die Auslage
* spielt selbst in diesem Durchgang nicht mit

Am Ende jedes Durchgangs werden die Punkte aller Spielenden notiert.

Jede gewonnene Karte zählt 1 Punkt.

Nach allen Durchgängen gewinnt die Person mit der höchsten Gesamtpunktzahl.
